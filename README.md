Log-Intention version 0.1
==========================

Demonstrate intention logging

INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

DEPENDENCIES

This module requires these other modules and libraries:

 Moose, Try::Tiny
